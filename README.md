# CasaClara

Landing page para **Casa Clara**, una pastelería artesanal enfocada en postres caseros, cuchareables y mesas dulces personalizadas.

## Contenido de la web

La web incluye:

- Presentación de marca y propuesta de valor.
- Acceso directo para pedidos por WhatsApp.
- Enlace a Instagram.
- Secciones de líneas de producto (Clásica y Especial).
- Información sobre cuchareables, mesas dulces y propuesta celebración.
- Horarios de envío.

## Ejecutar localmente

Este proyecto es estático (HTML + CSS). Para verlo:

1. Abrí `index.html` en tu navegador, o
2. Levantá un servidor local, por ejemplo:

```bash
python3 -m http.server 8080
```

Y luego entrá en `http://localhost:8080`.
